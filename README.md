# sigalih
SIGALIH merupakan kepanjangan dari **Sistem Informasi Desa Sirnagalih** (Website Desa) yang saya buat sebagai tugas akhir/skripsi saya di Informatika untuk menempuh gelar S1.
Fitur dalam Website ini yaitu Pembuatan Surat yang bisa sampai puluhan jenis surat yang disediakan. Selain pembuatan surat secara otomatis, terdapat juag fitur bank sampah.
